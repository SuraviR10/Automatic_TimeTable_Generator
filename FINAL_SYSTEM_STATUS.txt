================================================================================
SUPPORT & TROUBLESHOOTING
================================================================================
1. Server won't start:
   - Check Python version (py --version)
   - Reinstall packages (pip install -r requirements.txt)
   - Check port 5000 (netstat -ano | findstr :5000)

2. Timetable generation fails:
   - Verify all subjects have weekly_hours set
   - Ensure enough time slots available
   - Check lab rooms exist for department
   - Verify faculty assigned to all subjects

3. Year-semester dropdown not working:
   - Clear browser cache (Ctrl+Shift+Delete)
   - Hard refresh page (Ctrl+F5)
   - Check browser console for errors (F12)

4. Back button not working:
   - Clear browser cache
   - Ensure ?dept= parameter in URL
   - Check console for errors
