- timetable-new.htm ✅

CODE:
-----
document.getElementById('yearSelect').onchange = function() {
    const year = this.value;
    const semSelect = document.getElementById('semesterSelect');
    semSelect.innerHTML = '<option value="">--Select--</option>';
    if (year) {
        const sems = {'1':['1','2'],'2':['3','4'],'3':['5','6'],'4':['7','8']}[year];
        sems.forEach(s => {
            const opt = document.createElement('option');
            opt.value = s;
            opt.textContent = 'Semester ' + s;
            semSelect.appendChild(opt);
        });
    }
};

MAPPING:
--------
Year 1 → Semester 1, 2
Year 2 → Semester 3, 4
Year 3 → Semester 5, 6
Year 4 → Semester 7, 8

SOLUTION:
1. Check browser console (F12) for errors
2. Verify currentDepartment variable is set
3. Check if dept parameter is in URL
4. Clear browser cache (Ctrl + Shift + Delete)
5. Restart browser

ISSUE: Back button goes to wrong page
SOLUTION:
1. Check localStorage: localStorage.getItem('currentDepartment')
2. Should show department code (e.g., "ISE")
