TROUBLESHOOTING
===============
Problem: Server won't start
Solution: py --version (check 3.11+), pip install -r requirements.txt

Problem: Timetable generation fails
Solution: Check all subjects have weekly_hours, verify faculty assigned

Problem: Dropdown not working
Solution: Clear browser cache (Ctrl+Shift+Delete), hard refresh (Ctrl+F5)

Problem: Back button not working
Solution: Clear cache, check URL has ?dept= parameter
