COMPLETE WORKFLOW
================================================================================

FOR DEPARTMENTS:
----------------

1. LOGIN
   - Go to http://localhost:5000
   - Click "Department Login"
   - Enter department code (e.g., ISE)

2. SUBJECT DATABASE
   - Click "Subject Database" from dashboard
   - Select Academic Year (e.g., 2023-2024)
   - Select Year (1, 2, 3, or 4)
   - Select Semester (automatically populated based on year)
   - Add subjects with:
     * Subject Code
     * Subject Name
     * Credits
     * Type (Theory/Lab/Free)
     * Weekly Hours
     * Cross-Department flag (if applicable)
     * Open Elective flag (if applicable)
   - Click "Add Subject"
   - Repeat for all subjects

3. FACULTY CORE
   - Click "Faculty Core" from dashboard
   - Enter faculty details:
     * Name
     * Initials
     * Designation
   - Click "Add Faculty"
   - Repeat for all faculty

4. LAB MANAGEMENT
   - Click "Lab Management" from dashboard
   - Add lab rooms:
     * Room Code (e.g., ISE-LAB-01)
     * Lab Name (e.g., Database Lab)
     * Capacity (e.g., 30)
   - Click "Add Lab Room"

5. TIMETABLE GENERATOR
   - Click "Timetable Generator" from dashboard
   - Configure:
     * Academic Year
     * Year
     * Semester
     * Number of Sections
   - Click "Load Subjects"
   - Assign faculty to each subject for each section
   - Click "Generate Timetable"
   - Wait for generation (automatic retry up to 5 times)

6. VIEW & FINALIZE
   - Review generated timetable
   - Use "Smart Swap" if changes needed
   - Export to PDF
   - Click "Finalize" to save

================================================================================
ALL ENFORCED CONSTRAINTS
================================================================================

CONSTRAINT 1: Strict Weekly Hours
----------------------------------
Each subject placed EXACTLY as per weekly_hours setting
Example: DBMS with 4 weekly hours = placed exactly 4 times

CONSTRAINT 2: Daily Diversity
------------------------------
Same subject CANNOT appear twice on same day
Example: Monday cannot have DBMS in both P1 and P3

CONSTRAINT 3: Lab Continuity
-----------------------------
Labs ALWAYS in 2 consecutive periods
Labs NEVER cross breaks
Valid slots: P1-P2, P3-P4, P5-P6
Invalid slots: P2-P3 (crosses tea break), P4-P5 (crosses lunch)

CONSTRAINT 4: NSS/FREE → P6 Only
---------------------------------
NSS and FREE subjects ONLY in last period (P6)
Placed once per week

CONSTRAINT 5: Global Faculty Conflict
--------------------------------------
Faculty cannot teach two classes simultaneously
Checked across ALL departments and sections

CONSTRAINT 6: Lab Room Clash Prevention
----------------------------------------
Same lab room cannot be used by multiple sections at same time
Department-specific lab rooms tracked globally

CONSTRAINT 7: Open Elective Same-Slot
--------------------------------------
All OE subjects locked to SAME time slot across ALL departments
Admin sets the locked day+period

CONSTRAINT 8: Classroom Capacity
---------------------------------
Allocated room capacity >= section strength
System finds best-fit rooms to minimize wastage

CONSTRAINT 9: One Lab Per Day
------------------------------
Section cannot have multiple labs on same day

CONSTRAINT 10: Break Protection
--------------------------------
No class can span across tea break or lunch break

================================================================================
TROUBLESHOOTING
================================================================================

ISSUE: Server won't start
SOLUTION:
1. Check Python version: py --version (should be 3.11+)
2. Reinstall packages: pip install -r requirements.txt
3. Check port 5000: netstat -ano | findstr :5000
4. Kill process if needed: taskkill /PID <pid> /F

ISSUE: Timetable generation fails
SOLUTION:
1. Verify all subjects have weekly_hours set
2. Ensure enough time slots (5 days × 6 periods = 30 slots)
3. Check lab rooms exist for department
4. Verify faculty assigned to all subjects
5. System retries 5 times automatically

ISSUE: Faculty conflicts
SOLUTION:
1. Use "Smart Swap" in enhanced view
2. System suggests safe slots automatically
3. Check faculty timetable view

ISSUE: Year/Semester dropdown not working
SOLUTION:
1. Clear browser cache (Ctrl+Shift+Delete)
2. Hard refresh page (Ctrl+F5)
3. Mapping is: Year 1→Sem 1,2; Year 2→Sem 3,4; etc.

ISSUE: Back button not working
SOLUTION:
1. Clear browser cache
2. Ensure ?dept=XXX parameter in URL
3. Check browser console (F12) for errors
