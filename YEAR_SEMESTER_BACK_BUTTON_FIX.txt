1. YEAR-SEMESTER AUTOMATIC MAPPING
===============================================================================

ALREADY WORKING IN subject.htm:

When Year is selected, Semester dropdown automatically populates:

Year 1 → Semester 1, 2
Year 2 → Semester 3, 4
Year 3 → Semester 5, 6
Year 4 → Semester 7, 8

CODE IMPLEMENTATION:
--------------------
document.getElementById('yearSelect').onchange = function() {
  const year = this.value;
  const semSelect = document.getElementById('semesterSelect');
  semSelect.innerHTML = '<option value="">--Select--</option>';
  if (year) {
    const sems = {'1':['1','2'],'2':['3','4'],'3':['5','6'],'4':['7','8']}[year];
    sems.forEach(s => {
      const opt = document.createElement('option');
      opt.value = s;
      opt.textContent = 'Semester ' + s;
      semSelect.appendChild(opt);
    });
  }
};

✅ VERIFIED: Working correctly
